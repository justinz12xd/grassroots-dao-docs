## 📇 名刺デザイン（初期アウトプット）

この名刺は、草の根DAOの世界観をリアルの場でも伝えるために作成された初期ツールです。  
地域住民や協力者に向けて、活動の紹介・連絡先・支援導線（OFUSE）などを分かりやすくまとめています。

名刺の内容は「共感・自然・つながり・デジタル」の要素を盛り込み、DAOの価値観を実装した第一歩です。

### 🖼 ファイル構成

- `名刺表.pages`：活動名義・連絡先・支援QR付きの表面
- `名刺裏.pages`：草の根DAOの説明・連絡先付きの裏面
